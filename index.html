Work

Systopia Lab

2020/05 -

At the Systopia Lab, I am working on datacenter scheduling. Particularly, we are trying to improve revenue gain through increased spot instance (SI) utilisation. We can achieve this through a cooperative framework that communicates SI preferences to the VM scheduler, which allows us to improve SI scheduling with little to no cost to VM scheduling performance. Additionally, we investigated more intelligent eviction policies using machine learning. With predicted VM lifetimes, we can determine when a VM is expected to depart, which we can use in making scheduling decisions.
